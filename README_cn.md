# SDDS规范(beta)
  
快速构建二进制通讯解析程序的利器
  
[English document](https://github.com/byteferry/sdds)
  
## 简介

### 什么是SDDS?
    
* 数据结构
SDDS是"stream data dynamic structure"的缩写。
* DSL
SDDS是一种DSL (domain-specific language) .  
SDDS的schema不只是人可读，同时也是机器要理解的。   
* 解析引擎 
SDDS也指基于SDDS规范编写的解析引擎程序。用此，你可以快速完成你的二进制通讯的开发。
